## 📃 프로젝트 개요
- instagram SNS 클론코딩을 활용하여 같은 SNS 기반의 독서 경험 공유 플랫폼 형성 목적
- 좋은 글의 모음에는 좋은 사람들과 함께 해당 글에 대한 생각을 이야기할 수 있는 장으로서 활용될 수 있음
<br>

## 🛠 기술 스택
- Java, Spring boot, Spring Security
- JavaScript, HTML5, CSS3
- MariaDB, SQL
